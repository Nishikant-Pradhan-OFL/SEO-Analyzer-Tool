# 🔍 SEO Analyzer Tool

A simple tool to analyze key SEO elements of any webpage: title, description, headings, and more.

## ✨ Features
- Input any URL
- Displays meta tags, heading structure
- Works client-side or with Node.js

## 🛠️ Built With
- HTML, CSS, JavaScript
- Optional: Node.js + Puppeteer
